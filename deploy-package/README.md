# 电力交易AI平台 - 本地部署包
# ============================================

## 快速开始（推荐）

1. **双击 `一键启动.bat`** 即可运行！
2. 首次运行会自动安装Python依赖
3. 等待服务启动，浏览器会自动打开平台

## 访问地址

- **平台界面**: http://127.0.0.1:8000/console
- **API文档**: http://127.0.0.1:8000/docs

## 系统要求

- Windows 10/11
- Python 3.10 或更高版本

## 安装Python

1. 下载地址: https://www.python.org/downloads/
2. 安装时**务必勾选** "Add Python to PATH"

## 常见问题

### Q: 提示"不是内部或外部命令"？
确保Python已安装并添加到系统PATH。重启电脑后再试。

### Q: 依赖安装失败？
确保使用标准Python（从 python.org 安装），不是Microsoft Store版本。

## 停止服务

在命令行窗口按 **Ctrl + C** 即可停止服务。
