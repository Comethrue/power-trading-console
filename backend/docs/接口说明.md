# 接口说明（MVP）

## 通用返回结构

```json
{
  "success": true,
  "code": "OK",
  "message": "ok",
  "data": {},
  "request_id": "..."
}
```

## 鉴权

- 除 `GET /api/v1/health` 外，接口默认需要 `Authorization: Bearer <token>`。
- 支持 `POST /api/v1/auth/login` 使用用户名密码换取 JWT。
- 支持 `POST /api/v1/auth/refresh` 刷新 access token（refresh token 轮换）。
- 支持 `POST /api/v1/auth/logout` 撤销 refresh token。
- 默认令牌：
  - `admin-token`
  - `user1001-token`
  - `user1002-token`
- 默认登录用户：
  - `admin / admin123`
  - `user1001 / pass1001`
  - `user1002 / pass1002`

## 身份

- `POST /api/v1/auth/login`
- `POST /api/v1/auth/refresh`
- `POST /api/v1/auth/logout`
- `GET /api/v1/auth/whoami`
